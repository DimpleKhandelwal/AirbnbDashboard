# AirbnbDashboard

Global Airbnb Performance Dashboard
An interactive data visualization dashboard designed to analyze Airbnb listings, pricing trends, market distribution, and customer engagement across major global cities.

This dashboard provides a comprehensive overview of Airbnb’s global performance by exploring key metrics such as listings, hosts, reviews, pricing, and ratings. It highlights how major cities like Paris, New York, and Sydney dominate the market, while also uncovering trends in property types and pricing.

The dashboard enables users to:

Compare market share across top cities
Analyze growth trends in listings over time
Understand pricing differences by room type
Evaluate customer satisfaction through ratings

The dashboard was built using the following tools and technologies:

Power BI Desktop – Main platform for building interactive visualizations and reports
Power Query – Used for data cleaning, transformation, and shaping
DAX (Data Analysis Expressions) – Created calculated measures and dynamic insights
